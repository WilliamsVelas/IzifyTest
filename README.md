# IzyLite - Full-Stack Sales & Stock Management

IzyLite is a comprehensive solution designed to simplify business operations through efficient sales tracking and inventory management. This project consists of a high-performance backend API and a modern, intuitive mobile application.

---

## 🇺🇸 English

### Project Overview
The ecosystem is divided into two main components:
1.  **[Backend (izy-back)](./izy-back)**: A Node.js & TypeScript API that handles business logic, security, and data persistence with MongoDB.
2.  **[Frontend (izylite)](./izylite)**: A Flutter mobile application that provides a seamless user experience for managing products and processing sales.

### Core Features
- **Real-time Synchronization**: Frontend and Backend work together to keep stock levels updated.
- **Secure Authentication**: Protected routes using JWT and encrypted passwords.
- **Inventory Control**: Detailed management of products and categories.
- **Sales Flow**: Complete process from item selection to final sale and reporting.

### Global Stack
- **Frontend**: Flutter, BLoC, Shared Preferences.
- **Backend**: Node.js, Express, TypeScript, MongoDB, Mongoose.

- [Backend Setup Guide](./izy-back/README.md)
- [Frontend Setup Guide](./izylite/README.md)

## 🇪🇸 Español

### Descripción del Proyecto
El ecosistema se divide en dos componentes principales:
1.  **[Backend (izy-back)](./izy-back)**: Una API de Node.js y TypeScript que maneja la lógica de negocio, seguridad y persistencia de datos con MongoDB.
2.  **[Frontend (izylite)](./izylite)**: Una aplicación móvil en Flutter que ofrece una experiencia de usuario fluida para gestionar productos y procesar ventas.

### Características Principales
- **Sincronización en Tiempo Real**: El frontend y el backend trabajan juntos para mantener los niveles de stock actualizados.
- **Autenticación Segura**: Rutas protegidas mediante JWT y contraseñas encriptadas.
- **Control de Inventario**: Gestión detallada de productos y categorías.
- **Flujo de Ventas**: Proceso completo desde la selección de artículos hasta la venta final y reportes.

### Stack Global
- **Frontend**: Flutter, BLoC, Shared Preferences.
- **Backend**: Node.js, Express, TypeScript, MongoDB, Mongoose.
