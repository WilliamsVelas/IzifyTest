# IzyLite - Sales & Stock Control

IzyLite is a mobile application designed to simplify the management of sales and product inventory. It provides a robust and intuitive interface for business owners to track their daily transactions and keep an eye on their stock levels in real-time.

---

## 🇺🇸 English

### Features
- **Sales Management**: Process and record sales transactions efficiently.
- **Stock Control**: Maintain an updated list of products and their current quantities.
- **Modern Architecture**: Built with stability and scalability in mind using BLoC.

### Tech Stack
- **Framework**: [Flutter](https://flutter.dev/)
- **State Management**: [BLoC (Business Logic Component)](https://pub.dev/packages/flutter_bloc)
- **Networking**: [HTTP](https://pub.dev/packages/http)
- **Storage**: [Shared Preferences](https://pub.dev/packages/shared_preferences)
- **UI Components**: Flutter Material Design & SVG icons.

### Prerequisites
Before you begin, ensure you have the following installed:
- [Flutter SDK](https://docs.flutter.dev/get-started/install) (Stable version recommended)
- [Dart SDK](https://dart.dev/get-started/sdk)
- An IDE (Android Studio, VS Code, or IntelliJ)
- An Android Emulator or a Physical Android Device

### Installation & Setup
1. **Clone the repository**:
   ```bash
   git clone https://github.com/WilliamsVelas/IzifyTest.git
   cd izylite
   ```
2. **Install dependencies**:
   ```bash
   flutter pub get
   ```
3. **Run the application**:
   ```bash
   flutter run
   ```
